# **App Name**: Fortuna AI

## Core Features:

- Intelligent Conversational Workspace: A high-performance chat interface powered by the Gemini AI tool, supporting real-time messaging, markdown rendering, and persistent history storage in Firestore.
- Smart Text Summarization: Multi-mode summarizer tool offering quick, detailed, or bulleted distillations of large text blocks, with output saved directly to your workspace.
- AI Content Studio: A template-driven creation suite using AI tools to generate professional emails, essays, and social posts based on user parameters.
- Autonomous Prompt Improver: An optimization tool that takes raw user instructions and uses generative reasoning to rewrite them into high-fidelity engineering prompts.
- Integrated Analytics Engine: A data visualization dashboard providing insights into usage statistics, interaction history, and AI performance metrics stored in the database.
- Response Quality Feedback: A community-driven feedback system that allows users to rate AI responses, synchronizing metrics back to Firestore for analytics processing.
- Enterprise-Grade Auth & Sync: Seamless user onboarding via Firebase Authentication with multi-provider support and real-time state synchronization across devices.

## Style Guidelines:

- Background: #0B1020 (deep obsidian blue). Surface: #111827. Primary: #7C3AED (vibrant violet). Secondary: #06B6D4 (electric cyan). Accent: #F59E0B (amber). Text: #F9FAFB. All optimized for a professional dark-first theme.
- Headlines and body use 'Inter' for a clean, machinist look. Code blocks use 'Source Code Pro' for maximum legibility. Note: currently only Google Fonts are supported.
- Thin-stroke Lucide icons providing a surgical, minimalist aesthetic consistent with modern SaaS standards.
- A clean, collapsible sidebar architecture inspired by Vercel and Linear, utilizing glassmorphism effects and spacious, responsive grids.
- Silky-smooth transitions powered by Framer Motion, featuring subtle page fades, card hover scaling, and active-state button micro-interactions.
